Сервис предсказаний на основе ML‑модели
Описание
Это веб‑сервис для получения предсказаний с использованием обученной ML‑модели. 
Добавлена аутентификацию через JWT и минимальное разграничение прав к API.
Проект дополнен авторизацией на основе RBAC.

Основные возможности
/predict — получение предсказания по набору признаков;

/health — проверка работоспособности сервиса.

Запуск сервиса


Установка зависимостей
bash
pip install -r requirements.txt
Запуск
bash
uvicorn main:app --host 0.0.0.0 --port 8000
Сервис будет доступен по адресу: http://localhost:8000


Тестирование API
После запуска сервера:

Документация Swagger: откройте http://localhost:8000/docs  документация.

Проверка здоровья: GET http://localhost:8000/health
